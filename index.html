Transparent Tracking Pixel

A simple transparent 1x1 pixel image for tracking user visits and engagement on web pages.

Usage

Embed the tracking pixel in your Softr site using:

<img src="https://invite.disputedojo.com/pixel.png?source=email&campaign=launch" alt="" style="width:1px;height:1px;"/>

Tracking Parameters

You can add the following parameters to track different sources and campaigns:

Parameter Description Example
source Traffic source email, social, referral
campaign Campaign identifier spring2023, launch
medium Marketing medium newsletter, banner, cpc
content Content variant for A/B testing version_a, variant2
